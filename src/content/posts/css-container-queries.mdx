## 미디어 쿼리의 한계

사이드바 안에 있는 카드 컴포넌트가 있다고 합시다. 미디어 쿼리로는 뷰포트 너비만 알 수 있습니다. 사이드바가 접혔는지 펼쳐졌는지는 알 수 없죠.

## Container Queries

```css
.card-container {
  container-type: inline-size;
  container-name: card;
}

@container card (min-width: 400px) {
  .card {
    display: flex;
    flex-direction: row;
  }
}

@container card (max-width: 399px) {
  .card {
    display: flex;
    flex-direction: column;
  }
}
```

## 실전 활용

재사용 가능한 컴포넌트를 만들 때 특히 유용합니다. 디자인 시스템의 컴포넌트가 어디에 배치되든 적절하게 반응할 수 있습니다.
