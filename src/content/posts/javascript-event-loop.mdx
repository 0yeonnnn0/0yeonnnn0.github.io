## 들어가며

자바스크립트는 싱글 스레드 언어입니다. 그런데 어떻게 비동기 처리가 가능할까요?

## 콜 스택

함수가 호출되면 콜 스택에 쌓이고, 실행이 끝나면 제거됩니다.

```js
function first() {
  second();
  console.log('first');
}

function second() {
  console.log('second');
}

first();
```

## 태스크 큐와 마이크로태스크 큐

`setTimeout`은 태스크 큐에, `Promise.then`은 마이크로태스크 큐에 들어갑니다. 마이크로태스크 큐가 항상 먼저 비워집니다.

```js
console.log('1');
setTimeout(() => console.log('2'), 0);
Promise.resolve().then(() => console.log('3'));
console.log('4');
// 출력: 1, 4, 3, 2
```

## 정리

이벤트 루프는 콜 스택이 비어있을 때 큐에서 태스크를 꺼내 실행합니다. 이 단순한 규칙이 자바스크립트 비동기의 전부입니다.
