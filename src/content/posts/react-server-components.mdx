## SSR과 뭐가 다른가

SSR은 서버에서 HTML을 만들어 보내고 클라이언트에서 hydration합니다. RSC는 서버에서 실행되는 컴포넌트와 클라이언트에서 실행되는 컴포넌트를 명시적으로 구분합니다.

## 번들 사이즈의 혁명

서버 컴포넌트의 코드는 클라이언트 번들에 포함되지 않습니다. `moment.js`를 서버 컴포넌트에서만 사용하면 클라이언트에는 0KB입니다.

## 데이터 페칭의 변화

컴포넌트 안에서 직접 `await db.query()`를 할 수 있습니다. API 레이어가 사라지는 것이죠.

```tsx
async function PostList() {
  const posts = await db.query('SELECT * FROM posts');
  return posts.map(post => <PostItem key={post.id} post={post} />);
}
```
